#  Movie App (Ionic + Angular)

##  Project Overview
This is a mobile application built using Ionic and Angular that allows users to browse movies using The Movie Database (TMDb) API.  

Users can:
- View trending movies
- Search for movies
- View detailed movie information
- Add/remove favourites
- View cast and crew details

  

##  Features:-

###  Home Page:
- Displays today's trending movies from TMDb
- Search functionality to find movies by name
- Click on a movie to view details

###  Movie Details Page:
- Shows movie information:
  - Title
  - Overview (description)
  - Rating
  - Release date
- Displays cast and crew
- Button to:
  - Add to favourites
  - Remove from favourites
- Navigation:
  - Home button
  - Favourites button

###  Favourites Page:
- Displays all saved favourite movies
- Click a movie to view details
- Remove movies from favourites

###  Details Page (Cast & Crew):
- Displays selected person information:
  - Picture (or "No image" if not available)
  - Date of Birth (DOB)
  - Date of Death (DOD)
  - Also Known As (AKA)
  - Biography
- Shows other movies the person has worked in
